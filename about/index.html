Tab Dash is just a stupid little browser tab to display somewhat useful information. I built it as a side project. Mess around with the different background videos on the main page, the sounds, etc.

You can show / hide the UI on the home page by pressing "H".

Various bits of information can be copied to the clipboard by clicking on the clipboard icon next to the various bits of information.

This was simply a project I wanted to start so that I could continue honing my web development skills as most of the time I am a filthy network and systems architect.
